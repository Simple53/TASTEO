# 点点鲜 v1.0.0 发布说明

## 🎉 首次发布

**点点鲜** 是一个轻量级局域网点单系统,类似美团的点餐应用,适用于家庭聚会、小型餐厅或内部测试。

### ✨ 核心功能

#### 用户端
- 📱 响应式菜单展示
- 🛒 购物车管理
- 📝 订单提交
- 🎨 现代化 UI 设计

#### 商家端
- 📊 实时订单看板
- 📈 营业数据统计
- ✅ 订单管理
- 📥 数据导出 (Excel/CSV)

### 🚀 快速开始

#### 方式 1: 下载可执行文件 (推荐)
1. 下载对应平台的 Release 文件
2. 解压后双击运行
3. 访问显示的地址

#### 方式 2: 从源码运行
```bash
git clone https://github.com/yourusername/diandianxian.git
cd diandianxian
pip install -r requirements.txt
python init_data.py
python app.py
```

### 📦 Release 文件说明

- **diandianxian-v1.0.0-windows.zip** - Windows 可执行文件
- **diandianxian-v1.0.0-linux.tar.gz** - Linux 可执行文件
- **diandianxian-v1.0.0-macos.tar.gz** - macOS 可执行文件
- **diandianxian-v1.0.0-source.zip** - 源代码包

### 🛠️ 技术栈

- **后端**: Flask + SQLite
- **前端**: HTML5 + CSS3 + JavaScript
- **数据导出**: openpyxl + pandas
